# 我的 Cloudflare 页面

这是一个简单的静态网页模板，使用 HTML、CSS 和 JavaScript 构建。你可以将此模板部署到 Cloudflare Pages 上，并根据需要进行修改。

## 项目结构

- `index.html`：网页的主页面
- `styles.css`：页面样式
- `app.js`：动态显示当前时间的 JavaScript 脚本

## 部署到 Cloudflare Pages

1. 创建一个新的 GitHub 仓库，并上传此项目文件。
2. 登录到 Cloudflare Pages，连接 GitHub 仓库并部署。
3. 享受你的网站！
